# Inventory Management System (UAS Web Lanjut)

A simple CRUD (Create, Read, Update, Delete) inventory system built with **PHP**, **MySQL**, and **Bootstrap 5**.

## Features
- **User Authentication**: Secure Login and Registration.
- **CRUD Operations**: Manage inventory items (Add, Edit, Delete).
- **Session Management**: Prevents unauthorized access to the dashboard.
- **Responsive UI**: Built with Bootstrap 5 for a clean interface.

## How to Run
1. **Database**: Import `database.sql` into your MySQL server (Database name: `uas_web`).
2. **Configuration**: Check `db.php` and update MySQL credentials if necessary.
3. **Server**: Place the project in your web server root (e.g., `/var/www/html/`).
4. **Access**: Open `http://localhost/uas_project` in your browser.

## Tech Stack
- **Backend**: PHP 8.x
- **Frontend**: HTML5, Bootstrap 5
- **Database**: MySQL / MariaDB
